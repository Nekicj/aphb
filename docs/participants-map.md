# Интерактивная карта участников Almaty Physics Battles

Эта интерактивная карта визуализирует географию участников турнира Almaty Physics Battles, показывая города и страны, откуда приехали участники.

## Текущая статистика

На карте отображены участники из:
- **6 стран**: Казахстан, Кыргызстан, Таджикистан, Узбекистан, Беларусь, Россия
- **53 города** по всему региону
- **Общее количество участников**: 700+ человек

## Особенности карты

### Динамические маркеры
Размер и цвет маркеров зависят от количества участников в городе:
- **🔵 Очень темно-синий (30px)**: 100+ участников
- **🔵 Темно-синий (25px)**: 50-99 участников  
- **🔵 Синий (20px)**: 20-49 участников
- **🔵 Светло-синий (15px)**: менее 20 участников

### Интерактивность
- **Клик на маркер**: показывает всплывающее окно с подробной информацией
- **Легенда**: объясняет систему маркеров
- **Автоматическое масштабирование**: карта подстраивается под все точки

### Дизайн
- Адаптивный дизайн для всех устройств
- Стилизация в соответствии с общим дизайном сайта
- Плавные анимации и переходы

## Управление данными

### Добавление новых местоположений

#### Способ 1: Использование скрипта (рекомендуется)

```bash
node scripts/add-location.js "Название города" "Название страны" количество_участников широта долгота
```

**Пример:**
```bash
node scripts/add-location.js "Павлодар" "Казахстан" 35 52.2833 76.9500
```

#### Способ 2: Ручное редактирование

Отредактируйте файл `src/data/participants-locations.ts` и добавьте новый объект в массив `participantsLocations`:

```typescript
{
  city: 'Название города',
  country: 'Название страны',
  lat: широта,
  lng: долгота,
  participants: количество_участников
}
```

### Получение координат

Для получения координат городов можно использовать:
- [OpenStreetMap Nominatim](https://nominatim.openstreetmap.org/)
- [Google Maps](https://maps.google.com) (клик правой кнопкой → координаты)
- [Яндекс.Карты](https://yandex.ru/maps)

## Техническая информация

### Файлы компонента
- `src/components/participants-map.tsx` - основной React компонент
- `src/components/participants-map.css` - стили карты
- `src/data/participants-locations.ts` - данные о местоположениях

### Используемые технологии
- **Leaflet** - библиотека для интерактивных карт
- **OpenStreetMap** - источник картографических данных
- **React** - для компонента карты
- **Astro** - фреймворк сайта

### Особенности реализации
- Динамический импорт Leaflet для избежания SSR ошибок
- Клиентская инициализация с состоянием загрузки
- Кастомные маркеры с HTML содержимым
- Адаптивная легенда

## Обновление данных

После добавления или изменения данных:
1. Карта автоматически обновится при следующей загрузке страницы
2. Статистика в разделе "Масштаб и факты" обновится автоматически
3. Перезапуск сервера разработки не требуется
